# KSBÜ Mevzuat Bilgi Sistemi

Kütahya Sağlık Bilimleri Üniversitesi yönetmelik, yönerge ve senato kararları için modern mevzuat bilgi platformu.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env` to your Gemini API key
3. Run the app:
   `npm run dev`
